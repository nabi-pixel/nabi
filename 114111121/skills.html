HTML

75%

在學習 HTML 的過程中，我學會如何建立網頁的基本架構，包含標題、文字、圖片、連結以及表單等內容。透過實際製作網頁，我了解各種標籤的用途，也能將想呈現的資訊有條理地放在網頁上。（如下）

  • 建立網頁的基本架構與內容

  • 學習標題（h1~h6）、段落（p）、圖片（img）、連結（a）等標籤

  • 使用表單（form）製作資料輸入功能，例如文字輸入框、按鈕等

CSS

85%

學習 CSS 讓我知道如何美化網頁版面，例如調整字體、顏色、背景、間距及版面配置等。透過 CSS 的應用，我能讓原本單調的網頁變得更有設計感，也提升了網頁的閱讀體驗與整體美觀。（如下）

  • 美化網頁的整體外觀與風格

  • 調整字體大小、顏色、行距與間距

  • 設計背景顏色與圖片，提升視覺效果

  • 學習 box model（邊距 margin、內距 padding、邊框 border）

  • 使用 class 與 id 來控制不同區塊的樣式

Bootstrap

85%

在學習 Bootstrap 時，我認識到前端框架的便利性。利用 Bootstrap 提供的元件與版面系統，我可以更快速地完成網頁設計，並讓網頁能夠自動適應不同裝置的螢幕大小，製作出具有響應式設計的網站。

  • 使用現成元件快速設計網頁（按鈕、卡片、導覽列等）

  • 學習網格系統（Grid system）來做版面切割

  • 製作響應式網頁（RWD），讓網頁在手機、平板、電腦都能正常顯示

  • 減少手寫 CSS 的時間，提高開發效率

  • 學會套用 class 來快速完成美觀設計
